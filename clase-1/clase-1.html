Clase 1 - JS

La lectura del código JS es en cascada, es decir de arriba hacia abajo. Pero en el HTML, el script, va al final del body.
Para los comentarios, puede ser /* */ (en bloque) o // (comentario de una sola línea)

  • Es case-sensitive
  • Los espacios no cuentan, todo saldrá uno al lado del otro, como en HTML
  • Puede funcionar la línea de código con ; o sin él, pero es recomendable usarlo

Una variable es un espacio de memoria que reservamos para almacenar información importante durante la ejecución del programa. Las variables se declaran y luego se asignan

  • var se utilizaba para declarar variables en las versiones anteriores de JS, está obsoleto
  • let es lo recomendado
  • const es un tipo de variable que no varía en su valor, impidiendo que su valor se modifique después. Por convención en MDN, las const se escriben todas en Mayúsculas.

Operaciones matemáticas con JS

Cuando se realizan operaciones es importante usar paréntesis para conservar el orden de las operaciones, de lo contrario, hará la operación como se le plante.

Sentencias Prompt y Alert

La sentencia prompt mostrará un cuadro de diálogo para que el usuario ingrse un dato. Se puede proporcionar un mensaje que se colocará en el campo de texto.

  • Si el usuario ingresa un dato, se ingresa una string
  • Si el usuario cancela, me devuelve un null
  • typeof() me devuleve un string con el tipo de dato que fue almacenado en una variable
  • parseInt() me cambia un string por un número entero
  • parseFloat() me cambia un string por un número con decimales
