Condicionales

Si la variable contenida dentro del if entonces, ejecutará todo lo que está dentro de los curly brackets, si es falso no ejecutará ese pedazo de código y seguirá con el resto del código.

Se usa la palabra reservada if y evalúa una condición. Se utiliza else para ejecutar un código específico para cuando no se cumpla la validación del if.

Las variables booleanas pueden recibir el valor a partir de una evaluación booleana sobre otras variables.

Operadores Lógicos

Son comunes a todos los lenguajes de programación.

  • Para el operador igual, se utiliza ==, ya que sólo un signo igual se utiliza para asignar.
  • Cuando colocamos == va a comparar el valor, pero cuando colocamos === comparará el valor y el tipo de datos, es decir, es "estrictamente igual"
  • Así como el igual, existe el != distinto y el estrictamente distinto, !==
  • También es posible operar con menor, menor igual, mayor y mayor igual
  • Operador && se tienen que cumplir ambas condiciones.
  • Operador || se tiene que cumplir una u otra condición para que sea válido.
