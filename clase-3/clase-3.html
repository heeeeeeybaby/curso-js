Clase 3 - Ciclos

¿Qué es un ciclo?

Los ciclos son herramientas que nos permiten repetir un bloque de instrucciones cierta cantidad de veces. Se dividen en dos categorías:

  • Ciclos por conteo: Se repite un bloque de instrucciones una cantidad determinada de veces. Ejemplo: Ciclos for
  • Ciclos condicionales: Se va a repetir un bloque de instrucciones, siempre y cuando la condición de verdadero. Ejemplo: while y dowhile

Ciclo for

Generalmente lo voy a usar cuando sé la cantidad de repeticiones que debe tener mi coódigo. Se utiliza la palabra reservada for

Estructura

for (desde; hasta; actualización) {
código que se repite
}
  • Por convención, se utiliza la variable i para generar el ciclo for
  • El alcance de i es sólo dentro del ciclo for, es una variable auxiliar que nace y muere dentro de ese ciclo.
  • Cuando digo: i = i + 1 es lo mismo que decir: i++
  • Cuando digo: i = i - 1 es lo mismo que decir: --i
  • Sentencia Break: me permite interrumpir el ciclo.
  • Sentencia Continue: me permite saltar una iteración del ciclo, si se ocupa dentro de un if, se saltará el valor que cumpla con la condición del if.
De la documentación de MDN:
Si usamos la forma de sufijo, operador después del operando (por ejemplo, x-- ), el operador de disminución disminuye y devuelve el valor antes de disminuir.
Si se usamos la forma de prefijo, operador antes del operando (por ejemplo, --x), el operador de disminución disminuye y devuelve el valor después de disminuir.

Ciclos While

El ciclo while es un ciclo que se ejecuta mientras la condición evaluada sea verdadera.

  • Primero ejecutará lo que está en do y luego comprobará si es verdadero o falso la sentencia del while
  • Si la condición de while es falsa, ejecutará nuevamente lo indicado en la sección del do

Estructura

while(nombre != "salir"){
nombre = prompt("Ingrese un nombre: (Para salir ingrese salir)")
}

Ciclos Do While

Este tipo de ciclo, se diferencia del while porque ejecutará primero el código y después evalúa para saber si sigue ejecutando el código.

Estructura

do{
passUsuario = prompt("Ingrese su contraseña");
}while(passUsuario != passAutorizado);

Estructura Switch

Está especialmente diseñada para manejar de forma sencilla, múltiples condiciones sobre la misma variable. No es un ciclo.

  • Se utiliza la palabra reservada case para crear uno de los casos que suponemos ocurrirá, después de cada case, debemos agregar un break; sino ejecutará todos los casos hacia abajo.
  • Se utiliza el default para dar una respuesta a todos aquellos casos que no coincidan con las respuestas previstas anteriormente.

Estructura

switch (clima){
case "lluvia":
console.log("Mirá Netflix");
break;
}
