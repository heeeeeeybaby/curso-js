Clase 4 - Funciones

Una función es una serie de instrucciones que se agrupan para completar una tarea en particular

  • Una función debe tener sólo una función
  • Una función puede ser llamada desde cualquier parte del código, incluso antes de que esté declarada y funcionará igual.
  • Hay funciones que ejecutan una acción, y otras que retornan datos.
  • El nombre de la variable que va dentro del parámetro, puede ser cualquier nombre, y este sólo funcionará dentro de esa función

Estructura de una función

function nombreFuncion(parámetros){
// Set de instrucciones
}

¿Cómo se invoca una función?

nombreFuncion();

Scope o alcance de las variables

También llamado ámbito de las variables, es la zona del programa en la cual se define, el contexto al que pertenece

  • Alcance Global: Son variables que están declaradas fuera de una función o fuera de cualquier bloque de instrucciones que se cierra con llaves, lo que permite acceder a ella desde cualquier bloque de instrucciones.
  • Alcance Local: Cuando una variable es declarada dentro de una función, no es posible acceder a ella desde otro lugar del código.
  • Es posible repetir el nombre de una variable si una es global y otra es Local

Función Anónima

Es una función que se define sin nombre y se utiliza para ser pasada por parámetro o asignada a una variable

  • Las funciones pueden recibir como parámetros otras funciones

Estructura de una función anonima

let sumaAnonima = function (a, b){return a + b}
console.log(sumaAnonima(1000,1000));

Función flecha

Identificamos a las funciones flecha como funciones anónimas de sintáxis simplificada.

Estructura de una función flecha

let division = (a,b) => {
return a/ b;
}
console.log(division(100,10));

También es posible simplificarlo, eliminando el return, que queda implícito al eliminar las llaves

let multiplicacion = (a,b) => a * b; Las funciones pueden tener uno o más de 2 parámetros, hasta al rededor de 255 parámetros diferentes.
