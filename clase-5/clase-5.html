Clase 5 - Objetos

Un objeto es una colección de datos relacionados como una entidad, se componen de un listado de pares clave-valor, es decir tiene propiedades y valores

  • La gracia de los objetos es que me permite englobar un montón de propiedades y valores bajo una misma entidad
  • Se llama Objeto Literal cuando creamos un objeto escribiendo cada valor para cada propiedad "a mano".

Estructura

let alumnoSamuel = {
nombre: "Samuel",
apellido: "Tocaimaza",
edad: 35
};

Cómo accedemos a cada dato?

Utilizo el nombre del objeto y luego el . punto, y de esta manera accedo al valor de esa propiedad dentro del objeto seleccionado

console.log(alumnoSamuel.edad);

También es posible acceder a un valor utilizando corchetes e indicando el nombre de la propiedad dentro

console.log(perro["raza"]);

Asignar nuevos valores a propiedades

Para modificar un valor dentro de un objeto, simplemente me refiero a esa propiedad que quiero modificar e ingreso el nuevo valor tal cual se modifica una variable

auto.precio = 40000000;
  • Es posible modificar valores dentro de un objeto que sea asignado a una constante. Porque la referencia al objeto nunca se modifica pero sí sus propiedades, por lo que deberíamos trabajar los objetos como constantes
    • console.log(yerba);
      // 350
      yerba.precio = 400;
      console.log(yerba);
      // 400
Cuando trabajamos con valores: number, string, boolean, null. Estos se denominan valores primarios y accedemos directamente al valor. Pero, cuando trabajamos con Objetos, accedemos a una referencia de ese valor, por eso es que nos permite entrar y modificar las propiedades a pesar de declararse con const

Constructores

Es una función que usamos para crear un nuevo objeto cada vez que sea necesario, es como un molde.

  • Se utiliza el function para crearla y el nombre de esta, por convención lleva mayúscula la primera letra.
  • La palabra reservada this al crear un nuevo objeto, lo que estoy diciendo es referenciar a dónde se guardará la variable que se toma en la función
  • La palabra reservada new permite generar un nuevo objeto asignado a una constante, con sus debidos valores
  • Es posible pedir los valores del objeto mediante prompt
  • Las funciones en JS son generalmente de acceso global y los métodos son únicamente utilizados para ser invocados por los objetos que lo contienen.

Estructura

function Persona(nombre, apellido, edad){
this.nombre = nombre;
this.apellido = apellido;
this.edad = edad;
}
const persona1 = new Persona("Santiago", "Tocaimaza", 9); Cuando creamos funciones que trabajan directamente con los datos de un objeto, estas reciben el nombre de método

Clases

Las clases de JS, introducidas en la última versión, preveen de una sintaxis más clara y simple para crear objetos. Son un equivalente al empleo de la función constructora y permite definir distintos tipos de métodos.

Estructura

class Producto{
constructor(nombre, precio){
this.nombre = nombre;
this.precio = precio;
this.vendido = false;
} // Esto es un método dentro de la clase producto
sumaIva(){
return this.precio * 1.19.toFixed(2);
}
vender() {
this.vendido = true;
}
}
