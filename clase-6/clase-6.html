Clase 6 - Arrays

  • Puedo ir agregando valores a un array después de declararlo
  • En el array es posible almacenar cualquier tipo de dato.
  • Para organizarse mejor, los elementos de un array tienen un índice que los identifica. Comienza con el número 0, hasta el último elemento del array.
  • Es posible mostrar el array completo, o bien, sólo un valor de todo el array, para ello se escribe: console.log(nombreArray[índice]);
  • Nunca confundir el length con el índice

Modificar un elemento del array

Es posible modificar sólo un elemento del array, para eso necesito saber cuál es el índice del valor que quiero modificar y luego aplico la siguiente sintaxis: arrayLetras[2] = 100;

Recorrer un array

Es posible utilizar un ciclo for para recorrer el array completo

for(let i = 0; i < arrayLetras.length; i++){
alert(arrayletras[i]);
}

Métodos y Propiedades más comunes

  • Propiedad Length: Cuando lo pido en un array, me dice la cantidad de elementos
  • Método Push: Me permite agregar elementos por el final del array.La estructura es: nombreArray.push("Nuevo Valor");
  • Si quiero agregar elementos al inicio de un array, utilizo unshift La estructura es: nombreArray.unshift("Nuevo Valor");
  • Método Pop Me permite eliminar el último elemento del array. Este valor puedo almacenarlo en otra variable, por ejemplo: let eliminado = familia.pop();
  • Método Shift Lo utilizamos para eliminar el primer elemento del arrray, por ejemplo: let primerEliminado = familia.shift();
