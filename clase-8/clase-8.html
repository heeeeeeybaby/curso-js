querySelector

Este método nos permite seleccionar nodos con la misma sintaxis que usamos en CSS

  • Retorna el primero de los elementos que encuentra basado en la selección que le entregué
  • Para seleccionar todos los elementos que coincidan bajo la búsqueda, debemos utilizar querySelectorAll();

Modificar nodos

Es posible modificar nodos con las propiedades InnerText, InnerHtml, ClassName

  • innerText: Me permite acceder al contenido textual de algún elemento del DOM
  • innerHtml: Me permite definir el código html interno y generará nuevos nodos con su contendio dentro del elemento seleccionado.
  • className: Me permite agregar una nueva clase a un elemento

Agregar Nodos

Puedo ocupar el método createElement(); donde se agrega como parámetro, el nombre de la etiqueta html que necesito agregar

  • Para crear el nuevo elemento dentro de otro, debo pasar el elemento como const
  • Luego se crea el nuevo elmento en una constante nueva con createElement();
  • Para enlazar el nuevo elemento al interior del otro, se utiliza appendChild();

Eliminar Nodos

Se utiliza remove(); para eliminar un elemento del DOM.
